Group 18 Project : CAR RENTAL SYSTEM

------------------------------------Commands for Car Rental System------------------------------------------------
|                                                                                                                |
|         Create/Update via CSV file      : -enter                                                               |           
|         Insert Data via Cmd line        : -insert car_id car_name car_price car_mileage status(yes/no)"        |
|         Delete(All Cars)                : -deleteAll                                                           |
|         Delete(by car_id)               : -delete car_id                                                       |           
|         Update(by car_id)               : -update car_id car_name car_price car_mileage status(yes/no)         |   
|         Update Car Availability         : -updateStatus car_id status(yes/no)                                  |   
|         Print All cars                  : -printAll                                                            |           
|         Print by CarId                  : -print -id car_id                                                    | 
|         Print by CarName                : -print -name car_name                                                |
|         Search(for exact price)         : -get -price car_price                                                |           
|         Search(for exact mileage)       : -get -mileage car_mileage                                            |   
|         Search(by price (greater))      : -search -price -gt car_price                                         |           
|         Search(by price (less))         : -search -price -less car_price                                       |   
|         Search(by mileage (greater))    : -search -mileage -gt car_mileage                                     |
|         Search(by mileage (less))       : -search -mileage -less car_mileage                                   |       
|         Filter(in price range)          : -filter -price beg end                                               |   
|         Filter(in mileage range)        : -filter -mileage beg end                                             |   
|         Check Car Availability          : -check car_id                                                        |   
|         For Help                        : -h                                                                   |   
|                                                                                                                |
|                                                                                                                |
